# Smart Traffic Control System using MATLAB

## Overview
This project implements a smart traffic control system using MATLAB App Designer with image-based vehicle detection.

## Features
- Image processing-based vehicle detection
- Dynamic traffic signal control
- 4-way intersection simulation
- Real-time graph visualization
- Signal transitions (RED, YELLOW, GREEN)

## How it Works
- Four road images are processed using image processing techniques
- Vehicles are detected using thresholding and connected component analysis
- Traffic signals are dynamically controlled based on detected vehicle density

## Files Included
- app1.mlapp (Main MATLAB App)
- road1.jpg, road2.jpg, road3.jpg, road4.jpg (Input images)
- Report.docx (Project documentation)

## How to Run
1. Open the .mlapp file in MATLAB
2. Click Run
3. Click "Run Simulation" button
